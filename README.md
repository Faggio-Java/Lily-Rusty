Lily-Rusty - A binary based package manager for lfs written in rust
<br/> <br/>
installation:
```
sh install.sh
```
<br/> 
Usage: "lily $(option) $(package name)"

Dependencies {
<br/>Rust
<br/>Shell
<br/>Tar
<br/>}
<br/>

you can find all the available packages [here!](https://sourceforge.net/projects/bin-lily/files)
